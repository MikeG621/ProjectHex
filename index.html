ProjectHex

Author: Michael Gaisser Version: 0.0.4 Date: 130910

A project-based hex editor, with full tree navigation of binary files based on project files created by the user. The projects define the file structure and "building blocks" of the file, so defining a node once allows the use of the node throughout the entire binary.
